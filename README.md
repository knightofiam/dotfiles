# My dotfiles :)

## Installation

Download

```
curl -Ls https://github.com/knightofiam/dotfiles/archive/master.tar.gz | tar xz && mv -f dotfiles-master ~/dotfiles && cd ~/dotfiles
```

macOS

```
./install-macos.zsh
```

Linux
```
./install-linux.zsh
```
